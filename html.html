What I learned about HTML

Html is one language that is specifically used for coding and creaitng web pages. These web pages can be made to have headings, paragraphs to better organize information, and more graphics for visual aid such as tables, lists, photos, and videos. In addition to determining structure, a css page can be made to give a stylistic touch to the website, either by changing the color, font size or type, background and much more. Using this usually makes the website more aesthetically pleasing than the bare bones an html web page offers.
